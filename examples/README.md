## 📱 1. 發票掃描器 (`receipt_scanner.py`)

自動從發票圖片中提取關鍵資訊！

### ✨ 功能

- 💰 自動提取總金額
- 📅 識別日期
- 🏪 提取商家名稱
- 📦 列出商品項目
- 💾 儲存為JSON格式
- 📂 支援批次處理

### 🚀 使用方法

```bash
# 單個發票
python receipt_scanner.py receipt.jpg

# 批次處理
python receipt_scanner.py receipts/
```

### 📊 輸出示例

```

==================================================

📅 日期: 2024/12/15

   1. 御飯糰
   2. 茶葉蛋
   3. 咖啡
   4. 麵包
   5. 牛奶

```

## ⚡ 2. 性能基準測試 (`benchmark.py`)

測試並比較不同配置的處理速度和記憶體使用！

### ✨ 功能

- 📈 自動生成測試PDF
- ⏱️ 測量處理速度
- 💾 監控記憶體使用
- 📊 漂亮的結果表格（使用rich）
- 🎯 多種測試場景

### 🚀 使用方法

```bash
# 執行預設測試
python benchmark.py

# 自訂測試（未來支援）
python benchmark.py --mode structure --dpi 300
```

### 📊 測試場景

- 10頁 PDF, basic模式, DPI 150
- 10頁 PDF, basic模式, DPI 200  
- 10頁 PDF, hybrid模式, DPI 150
- 50頁 PDF, basic模式, DPI 150

## 🎨 3. CLI 美化模組 (`rich_ui.py`)

為命令列介面添加超炫視覺效果！

### ✨ 功能

- 🎨 彩色輸出
- 🖼️ ASCII Art Logo
- 📊 漂亮的結果表格
- 💫 進度條
- ✅ 狀態訊息（成功/錯誤/警告）
- 📈 性能摘要面板

### 🚀 使用方法

```python
from paddleocr_toolkit.cli.rich_ui import *

# 顯示Logo和橫幅
print_logo()
print_banner()

# 狀態訊息
print_success("處理完成！")
print_error("檔案不存在")
print_warning("DPI設定較低")
print_info("開始處理...")

# 結果表格
results = [
    (1, 145, 0.95),  # (頁數, 文字數, 信心度)
    (2, 132, 0.88),
    (3, 167, 0.92),
]
table = create_results_table(results)
console.print(table)

# 性能摘要
stats = {
    'total_pages': 3,
    'total_time': 12.5,
    'avg_time_per_page': 4.17,
    'peak_memory_mb': 245.8,
    'total_texts': 444
}
print_performance_summary(stats)
```

### 📊 視覺效果

- ✅ 綠色：成功訊息
- ❌ 紅色：錯誤訊息
- ⚠️ 黃色：警告訊息
- ℹ️ 藍色：資訊訊息
- 🎯 美化的統計表格
- 📈 專業的摘要面板
